# Design Reference Notes

This folder holds non-production design references used to guide implementation.

Current reference:

- `application_tracker_ui_mockup_inspectable.html`

How to use it:

- Treat the shell, backdrop, card hierarchy, radii, and split-layout patterns as authoritative reference direction.
- Treat icons, illustrative placeholder content, and notification or avatar affordances as illustrative only.
- Production implementation should soften decorative excess and keep workflow clarity, accessibility, and product semantics intact.

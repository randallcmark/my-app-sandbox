# Application Tracker UI Handoff

## What this package contains

- `index.html`  
  A desktop mockup reconstruction that gives engineering a real DOM structure to work from.

- `styles.css`  
  Split CSS for layout, surfaces, components, spacing, and responsive behaviour.

- `design_tokens.json`  
  Inferred design tokens and layout constants.

- `component_notes.md`  
  Product and engineering guidance for turning the mockup into reusable components.

- `ui_contract.json`  
  A compact machine-readable summary of the surface model and interaction intent.

- `components/README.md`  
  Suggested future component-level decomposition.

## Recommended use

Use `docs/ui/application_tracker_ui_mockup_inspectable.html` as the high-level reference.
Then use the files in this folder to guide implementation work.

## Caveat

This is a reconstruction for development guidance. It is not a direct export from a source design tool.

The original generated image is intentionally not embedded in the repo copy to keep the handoff text-first and Git-friendly.
